1.2 - 通用搜索 ​

1.2.1 - 常用引擎 ​

  • Google 谷歌👍（世界第一大搜索引擎）
  • Bing 必应👍（世界第二大搜索引擎，国内打开会强制跳转CN）
  • BingCN 必应中国（国内版必应，搜索结果有审查）
  • 百度（国内第一大搜索引擎）
  • 搜狗（国内第二大搜索引擎）
  • 360
  • 神马 (夸克)

1.2.2 - 区域引擎 ​

  • 🇷🇺Yandex（支持中文搜索）
  • 🇺🇸Yahoo 雅虎（支持中文搜索）
  • YahooTW 雅虎台湾 (奇摩)（支持中文搜索）
  • 🇷🇺Rambler（既不支持中文搜索也不支持英文搜索）
  • 🇺🇸Ask（支持中文搜索）
  • 🇫🇷Qwant（支持中文搜索）
  • 🇩🇪Fireball（支持中文搜索）
  • 🇰🇷NAVER（支持中文搜索）
  • 🇺🇸Brave

1.3 - 聚合搜索 ​

  • 虫部落快搜👍
  • 多搜搜
  • 轻略搜索
  • 书签库
  • 小不点（只提供正规网站索引）
  • Baidugle (谷百度歌)👍
  • SearXNG

1.3.1 - 新标签页 ​

  • iTab 新标签页
  • CSDN 搜索
  • Infinity Tab

1.4 - 特色搜索 ​

  • Ecosia（公益搜索引擎，广告收入将捐给 WWF 基金会）
  • Lumen（谷歌删除内容搜索，搜索因 DMCA 被删除的谷歌搜索结果，通过邮箱获取）
  • Dataset Search（数据集搜索引擎，谷歌 2020 年上线的数据集搜索引擎）

1.4.1 - 网页存档 ​

  • Wayback Machine👍（存档查询，网页时光机，自动存档网页备份）
  • Webpage archive（存档工具，网页时光机，手动存档网页备份）
  • 网页快照网（快照查询，快速查询多家搜索引擎的网页快照）

1.4.2 - 元搜索引擎 ​

整合多个搜索引擎的结果，然后再将返回给用户。

  • Infospace👍
  • WebCrawler Search

1.4.3 - 匿踪搜索 ​

  • Backdata
  • Startpage
  • DuckDuckGo👍

1.4.4 - 儿童搜索 ​

会过滤掉不适合儿童访问的内容。

  • Kiddle
  • KidRex
  • KidzSearch

1.5 - 垂直搜索 ​

1.5.1 - 专用搜索 ​

  • Wiby（远古网站搜索引擎，搜出来的结果都是 2000 年左右时的网页）
  • Filmot（字幕搜索引擎，可根据关键词自动搜索 U2B 视频字幕）
  • GifCities（远古网站搜索引擎，搜索出来的结果都是 2000 年左右时的 GIF 图片）
  • Listen Notes（播客搜索引擎，全球最好的播客搜索引擎）
  • Keyword Tool（搜索联想词查询，搜索国外各大平台的关键联想，可用于 SEO）

1.5.2 - 开发搜索 ​

  • Devv（AI 智能搜索，基于 AI 驱动的程序员搜索引擎）
  • Goobe（程序员搜索，一个好用的程序员搜索，支持多种类别的搜索）
  • Cursor（AI 智能搜索，基于 ChatGPT 的程序员搜索引擎）
  • grep.app（源代码搜索，从 GitHub 项目中搜索源代码）
  • aiXcoder（源代码搜索，从多个平台项目中搜索源代码）
  • 开发者搜索（程序员搜索，百度针对开发者群体开发的技术搜索引擎）
  • SearchCode（源代码搜索，从 4000 万个项目中搜索源代码）

## 介绍

基于 Clash Meta 的多平台代理客户端，简单易用、开源无广告。

Windows 桌面端展示：
![](https://github.com/chen08209/FlClash/blob/main/snapshots/desktop.gif?raw=true)

Android 移动端展示：
![](https://github.com/chen08209/FlClash/raw/main/snapshots/mobile.gif?raw=true)

- ✈️多平台：Windows、Android、MacOS、Linux；
- 💻自适应多个屏幕尺寸，多种颜色主题可供选择；
- 💡基本 Material You 设计, 类 Surfboard 用户界面；
- ☁️支持通过 WebDAV 同步数据；
- ✨支持一键导入订阅。

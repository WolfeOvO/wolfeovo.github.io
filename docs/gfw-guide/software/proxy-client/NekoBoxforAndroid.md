## 介绍

::: details 支持的代理协议
- SOCKS (4/4a/5)；
- HTTP(S)；
- SSH；
- Shadowsocks；
- VMess；
- Trojan；
- VLESS；
- AnyTLS；
- ShadowTLS；
- TUIC；
- Hysteria 1/2；
- WireGuard；
- Trojan-Go (trojan-go-plugin)；
- NaïveProxy (naive-plugin)；
- Mieru (mieru-plugin)。
:::

::: details 支持的订阅格式
- 一些广泛使用的格式，如 Shadowsocks, ClashMeta 和 V2RayN；
- Sing-box 出站。
仅支持解析出站（即节点），分流规则等信息会被忽略。
:::

请于[此处](https://matsuridayo.github.io/nb4a-plugin/)（[Naive Proxy（维护中）](https://github.com/klzgrad/naiveproxy/)、[Hysteria Plugin（停止维护）](https://github.com/MatsuriDayo/plugins/releases?q=Hysteria)）下载插件以获得完整的代理支持。

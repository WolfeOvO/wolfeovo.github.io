**流量成本=国内+国外**<br>
我们目前 90% 节点都是隧道接入，流量平均成本为**国内¥40+国外¥30=¥70/TB**。

**直连线路成本：国外落地节点**<br>
| 优点 | 缺点 |
| :--: | :--: |
| 成本低 | 线路决定质量，经常被墙，稳定性差 |

**中转线路成本：国内+国外落地节点**<br>
| 优点 | 缺点 |
| :--: | :--: |
| 通过汇聚流量再出国到海外节点稳定性与速度都得到一定提升 | 由于是国内汇聚流量再出口到海外依旧存在线路差的情况，容易被反向墙会死一大半节点 |

**隧道线路成本：国内+国外中转+国外落地节点**<br>
| 优点 | 缺点 |
| :--: | :--: |
| 与中转对比隧道加密流量汇聚到海外中转再分发到各节点海外稳定性与速率都得到了提升，同时也避免了反向墙的发生 | 成本高，线路增加故障率也会上升，国内端或国外端中转出现故障依旧会死一大片节点 |

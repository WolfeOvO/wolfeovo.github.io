# Telegram 客户端合集

::: tip
2021 年开始，Telegram 限制第一次注册必须使用官方手机客户端才能发送验证码到短信，桌面端和第三方客户端不能直接注册。
:::

## 官方下载

Telegram 官网：[https://telegram.org/](https://telegram.org/)<br>
官方下载页：[https://telegram.org/apps](https://telegram.org/apps)

- **手机端**：[Android](https://telegram.org/android)、[iPhone & iPad](https://telegram.org/dl/ios)
- **桌面端**：[Windows/Mac/Linux](https://desktop.telegram.org/)、[MacOS](https://macos.telegram.org/)
- **Web 应用**（较少用）：[WebK](https://telegram.org/dl/webk)、[WebA](https://telegram.org/dl/weba)

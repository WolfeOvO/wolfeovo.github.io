6.4 - 音乐播放器 ​

6.4.1 - Windows ​

  • MPC-HC GitHub - clsid2/mpc-hc: Media Player Classic（纯碎的、小巧的、极致快速的音乐播放器，MPC-HC 的 Fork 版本）
  • MPC-BE GitHub - Aleksoid1978/MPC-BE: MPC-BE – универсальный проигрыватель аудио и видеофайлов для операционной системы Windows.（同上，MPC-HC 的 BE 版本）
  • Foobar2000👍（插件丰富，支持高度定制化的音乐播放器，推荐配置 Foobox）
  • VLC👍（开源的多媒体播放器，支持几乎所有音视频格式，全平台支持）
  • AIMP（来自俄罗斯的一款闭源多媒体播放器，支持 Win、安卓）
  • Winamp（化石级播放器，现已重新回归，并拥有了全新的设计，仅支持 Windows）

6.4.2 - MacOS ​

  • Foobar2000👍（支持高度定制化的音乐播放器，现在也支持 Mac 版本了）
  • IINA👍（可能是 Mac 最受欢迎的多媒体播放器，开源免费，简洁美观）
  • LyricsX GitHub - ddddxxx/LyricsX: 🎶 Ultimate lyrics app for macOS.（Mac 下的歌词辅助工具，支持搜索和滚动显示歌词，支持本地任何播放器）
